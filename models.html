Welcome to the ACTIVE Data Model Repository

Reusable objects and object definitions are listed below

  • Gateway System

    Gateway Application Object describing base system settings

  • Gateway Wireless Interface

    Gateway Application Object for representing wireless IP interfaces

  • Gateway Fixed Interface

    Gateway Application Object for representing fixed IP interfaces

  • Log Control

    Controls the log handling for the sensor data

  • EMM Integrator

    Object for defining the components of an Energy Measurement Module

  • Device Binding

    Connects the EMM to the device it measures

  • BLE GATT Proxy

    Generic object mapping BLE services via low level operations over LWM2M. Each object represents a single BLE service

  • PAN Interface

    Generic object for modelling endpoint or gateway PAN interfaces, such as Bluetooth, BLE and Zigbee

  • Native Operations

    Generic object that is one of several objects specified to provide the LWM2M server with hypermedia-like operations on the gateway

  • UCI Commands

    Generic object that allows LWM2M to issue system level calls to OpenWRT's UCI configuration system

  • BLE Commands

    Generic object that allows LWM2M to issue system level BLE operations via a bluetooth interface on a gateway or device
