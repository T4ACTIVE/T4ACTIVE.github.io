Resource ID Access Type Multiple Instances Mandatory Type Range/Enumeration Units Description
DeviceName 20190 R Single Mandatory String / / Name of the device under measurement, e.g. "Wireless Access Point".
DeviceID 20191 R Single Mandatory String / / Identifier for the device under measurement, e.g. serial number, IP or MAC address.
DevicePhysLoc 20192 R Single Mandatory String / / The physical location of the device, e.g. floor/corridor/room.

Example usage

The device binding object is used to identify which physical device the EMM is connected to and measuring. The resources include options to identify the device using name, ID (such as MAC or Serial) and/or location.
